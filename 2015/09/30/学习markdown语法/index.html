学习markdown语法

发表于 2015-09-30   |   分类于 杂记   |  

详见内

1.标题

三个或更多=就是大标题,===放标题下面

大标题例子
===

三个或更多-就是小标题

小标题例子
---

一级标题用1个

# 我是一级标题(#后有个空格)

二级标题用2个

## 二级标题用2个#

and so on

###...依次类推吧,一共只有6级

2.字体

斜体

将需要设置为斜体的文字两端使用1个“*”或者“_”夹起来,注意符号与字之间不要有空格

*斜不斜*
_斜不斜_

斜不斜
斜不斜

粗体

将需要设置为斜体的文字两端使用2个“*”或者“_”夹起来

**粗不粗**
__粗不粗__

粗不粗
粗不粗

3.无序列表

一个 -/+/* (符号后面有空格哦)后面拼接一段话,就是一个无序列表

- 无序例子1
+ 无序例子2
* 无序例子3

4.有序列表

有序列表大致同无序列表,不过是用数字哦

1. 数字不能省略但是可以无序,老规矩(点号后面的空格不能省略)
2. 同上

5.嵌套列表

嵌套列表,无非就是前面+个空格

- 嵌套1层,用- 做前缀,注意空格
 + 嵌套2层,是起一行,在符号面前加一个空格,再用+ (注意还是有空格)
 + 嵌套2层,同上
  - 嵌套3层了,同上,符号前面在上面的基础上再加一个空格
- 还是嵌套1层,和第一个一样就可以了

5.我是漂亮的分割线~线~线~线~线~线~线~线~

---  直接用三条-,就是分割线,后面不要加任何东西

6.文字超链接

直接上文字,要显示的文字用[]包起来,后面紧跟一个(),(里面第一个参数写链接地址,然后空格 ,打””,”里面当然写上tip”)

我是屌屌的超链接

[我是屌屌的超链接](http://kevinmky.github.io/ "KevinMKY的新博客哟~客官欢迎常来哟")

7.直接显示链接地址

直接上文字,<>内些链接就可以了

http://kevinmky.github.io/

<http://kevinmky.github.io/>

8.图片

A.不指定图片大小,直接又大多放多大(就是任性).
图片地址可以接外部地址,也可以是文件内地址

![这里是图片下方的描述](/path/to/img.jpg)

B.指定大小的用IMG语法

<img src="/images/avatar.jpg"  width="128" height="128" alt="黑眼圈的猫咪" />

9.标记代码块

在每行想要当成代码块显示的文字前加上四个空格,或者按一下tab即可,记得和上一行的文字要隔开(打个回车)

空格 空格 空格 空格 code
或
TAB code

想要给一大段代码标记,2个方法

A.整段代码前后分别加

<pre><code>
整段代码,效果如下,没有高亮
</code>\</pre>


- (void)textFiledEditChanged:(NSNotification *)obj
{

UITextField *textField = (UITextField *)obj.object;

}

B.每一行都加4空格或TAB,就会高亮显示部分

- (void)textFiledEditChanged:(NSNotification *)obj
{

    UITextField *textField = (UITextField *)obj.object;

}

C.用三个`上下夹击一整段代码,`之间不要有空格

` ` `
- (void)textFiledEditChanged:(NSNotification *)obj
{

    UITextField *textField = (UITextField *)obj.object;

}
` ` `

效果如下(推荐这种,可读性更强,使用也便捷)

10.注释

A.块注释

  • 通过在文字开头添加“>”表示块注释。（当>和文字之间添加五个blank时，块注释的文字会有变化。）

我就是屌屌的块注释

我就是屌屌的块注释

\>我就是屌屌的块注释     (忽略前面那个\,只需要一个>即可)

\>     我就是屌屌的块注释     (此处有5个空格,忽略前面那个\,只需要一个>即可)

B.只有作者可见的注释,用下面这个<!-- 注释 -->即可

<!-- 注释 -->

11.转义

加上一个\即可

Markdown中的转义字符为\，转义的有：
\\ 反斜杠
\` 反引号
\* 星号
\_ 下划线
\{\} 大括号
\[\] 中括号
\(\) 小括号
\# 井号
\+ 加号
\- 减号
\. 英文句号
\! 感叹号
